== Description ==

this plugin allows to assign categories to users.

view, edit and delete posts only categories selected for user

------
Italiano

questo plugin permette di assegnare agli utenti registrati del vostro blog determinate categorie.

Permettendo quindi di manipolare i post solo di determinate categorie da voi scelte

== Installation ==


1. Download plugin

2. Copy to your wp-content/plugins directory

3. Activate within wordpress admin

4. Go to users -> Assign users categories

5. select a users and select the categories...
